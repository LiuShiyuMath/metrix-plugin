How to re-verify this page

Run the judge harness at the worktree root:

bash bin/judge.sh
cat judge.json | claudefast --plugin-dir /tmp/empty

The harness re-runs the stream-json probe, the A/B plugin-dir probe, the jq schema check, the grep anchor check on this page, and the node syntax check on every hook. The final verdict is made by a clean claudefast process with no plugins loaded, so the agent under test cannot grade its own homework.
